# Recent Adaptation in Lake Malombe C.mloto — Analysis Code Archive

This repository contains reproducible code and analysis notebooks accompanying the study and manuscript on **recent adaptation and fisheries-induced evolution in _Copadichromis mloto_** populations from Lake Malombe and Lake Malawi.

All analyses were conducted in Python and R using publicly available tools for population genomics, transcriptomics, and simulation-based inference.

```
Code_archive_mss/
├── Notebooks/ 
│   ├── recent_adaptation_Malombe_manuscript_code_pub.ipynb # Main manuscript analysis pipeline
│   ├── simulation_output_pub.ipynb # Explorations of coalescent simulation output
│   └── pyDeseq_pub.ipynb # Workflow automations referenced in notebooks.
├── Snakemake/ # Workflow automations referenced in notebooks.
└── README.md
```
